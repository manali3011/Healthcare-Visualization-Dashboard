# Healthcare Visualization Dashboard

## Project Overview

This project presents an interactive healthcare dashboard developed using Power BI.

The dashboard provides insights into patient admissions, medical conditions, hospital performance, and billing trends through interactive visualizations.

## Dashboard Components

### KPI Cards
- Total Patients
- Total Billing Amount
- Average Billing Amount

### Charts
- Patient Admissions Over Time (Line Chart)
- Patients by Medical Condition (Bar Chart)
- Billing Amount by Hospital (Column Chart)

### Filters/Slicers
- Gender
- Admission Type

## Tools Used

- Power BI
- Microsoft Excel / CSV
- GitHub

## Dataset

Healthcare Dataset (Cleaned)

## Key Insights

- Track patient admission trends over time.
- Identify the most common medical conditions.
- Compare billing amounts across hospitals.
- Filter data dynamically using slicers.
